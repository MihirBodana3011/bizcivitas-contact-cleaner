# Bizcivitas Contact Cleaner

A premium contact cleaning tool for Bizcivitas. Clean your CSV/Excel contact lists, format Indian mobile numbers, and remove entries without phone numbers effortlessly through a beautiful glassmorphic web interface.

## 🚀 Features
- **Serverless & Fast**: Processes everything in your browser—no file uploads to any server!
- **Smart Cleaning**: Formats Indian mobile numbers and filters out invalid entries.
- **Premium UI**: Modern glassmorphic design.
- **Privacy First**: Your data never leaves your computer.

## 🛠️ Static Hosting (GitHub Pages)
This app is built with pure HTML, CSS, and JS (using SheetJS). It is designed to be hosted on **GitHub Pages**.

## 📁 Project Structure
- `index.html`: Main page.
- `static/style.css`: Premium styles.
- `static/script.js`: Core logic using SheetJS.
- `Logo/`: Brand assets.
