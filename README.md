# Bizcivitas Contact Cleaner

A premium contact cleaning tool for Bizcivitas. Clean your CSV/Excel contact lists, format Indian mobile numbers, and remove entries without phone numbers effortlessly through a beautiful glassmorphic web interface.

## 🚀 Features
- **Smart Cleaning**: Automatically formats and validates Indian mobile numbers.
- **Auto-Filter**: Removes any contact lacking a valid phone number.
- **Premium UI**: Modern, glassmorphic design based on the Bizcivitas brand colors.
- **Custom Output**: Automatically names processed files as `[Name] + Bizcivitas Contact`.
- **Supports**: `.csv`, `.xlsx`, `.xls`

## 🛠️ Setup & Installation

1. **Clone the repository**:
   ```bash
   git clone [your-github-repo-url]
   cd bizcivitas-contact-cleaner
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the app**:
   ```bash
   python app.py
   ```

4. **Access in Browser**: Open `http://127.0.0.1:5000`

## 📁 Project Structure
- `app.py`: Flask backend.
- `Contact_Cleaner_Tool.py`: Core cleaning logic.
- `templates/`: HTML interface.
- `static/`: CSS, JS, and Logo.
- `requirements.txt`: Python package requirements.
